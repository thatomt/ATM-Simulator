# ATM-Simulator

A basic command-line banking application written in Python that allows users to check their balance, deposit money, and withdraw funds.



# Overview

This project simulates a simple banking system that runs in the terminal. It allows users to perform basic financial operations through a menu-driven interface.

The application maintains a balance and updates it based on user actions.

Features

1.Check current account balance

2.Deposit money into the account

3.Withdraw money from the account

4.Prevent withdrawals that exceed the available balance

5.Simple command-line interface


# How It Works
The program starts with an initial balance of 1000. Users interact with the system through a menu that provides several options.

The application continues running in a loop until the user chooses to exit.

Available operations include:
1. Check Balance

2. Deposit Money

3. Withdraw Money

4. Exit the Application
